# v0.5.1 Release Notes

`v0.5.1` is a patch release focused on frontend packaging consistency in cluster deployments.

## Highlights

- Runtime image now builds and includes Workbench frontend assets.
- `agent.ui` package data explicitly includes `frontend_dist` static files.
- Added `.dockerignore` rules to avoid stale local frontend artifacts entering release images.

## Validation

- `python3 -m pytest -q`
- `PYTHONPYCACHEPREFIX=/tmp/pythoncache python3 -m compileall agent tests`
- `docker build -t k8s-diagnosis-agent:test-frontend-dist .`
- Verified in-container assets:
  - `/usr/local/lib/python3.11/site-packages/agent/ui/frontend_dist/index.html`
  - `/usr/local/lib/python3.11/site-packages/agent/ui/frontend_dist/assets/*`
